# Frontend Fundamentals 모의고사

## Getting started

```sh
yarn dev
```

## Goal

[ ] 적금 상품 목록 API 연동 및 출력

[ ] 저축 목표 입력 기능

[ ] 적금 상품 목록 선택 기능

[ ] 계산 결과 탭

 - 예상 수익 금액
   - `월 납입액 * 저축 기간 * (1 + 연이자율 * 0.5)`
 - 목표 금액 차액
   - `목표 금액 - 예상 수익 금액`
 - 추천 월 납입 금액 
   - `목표 금액 ÷ (저축 기간 * (1 + 연이자율 * 0.5))`
   - 1,000원 단위로 반올림
[ ] 추천 상품 목록
 - 이자율 높은 순으로 상위 2개 추천
 - 추천 상품 클릭 시 선택 기능
